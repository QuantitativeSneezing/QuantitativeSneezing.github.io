Selected projects

Storm Is a full stack clone of Steam, the e-commerce gaming platform. I wanted to clone steam since it's an interesting mix of both an e-commerce platform- with its initial (and still primary) feature being an online video game storefront, and a social network, given the rich community and friend features added by Valve over time. As of the latest update of this website, the live Storm site currently has the ability to buy (unplayable) fake games, as well as leave real reviews on those games. Planned features are the ability to make friends with other users, and give them unique nicknames; the ability to send, edit and delete direct messages to users who are your friends; the ability to wishlist and search for games you might enjoy; and (if I get overambitious) the ability to make community groups with members in it and socialize with other Storm users.

Storm

Storm was built using a React frontend and a Flask/SQLalchemy backend. I'm quite fond of python in general (Features like list comprehension are quite nice), although it was a particularly nice language to use here because to use flask because of how easily I could match my API routes to my intended redux state shape during the project.

Eriscord

Is another React/SQLAlchemy project, this time made in a group setting. We decided to clone Discord to get some experience with websockets and live updating information. I found that aspect quite fun, and learning the new tech was nice. Group collaboration was also a good experience- I learned quite a lot from my teammates, particularly how to efficiently break up pages into many React components to make workflow easier.Functionality wise, the website allows creation of servers and channels within said servers, as well as live messaging through both servers and private DMs with other users.

AirDnd

Was originally going to be a purely fantasy themed Airbnb clone, although as time went on it slowly took on a significantly more sci-fi theme in terms of the locations. This one's actually pure Javascript- the front end is still React, but for the backend I used Express/(Sequelize on local, postgres on live). In terms of functionality, the website allows users to host their own spots as well as post review for existing spots. Planned features include a search bar and the ability to book spots.
